## 4. Natureza gerada de `routeTree.gen.ts`

`src/routeTree.gen.ts` é gerado e mantido pelo plugin TanStack React
Start (Vite) ativo no ambiente Lovable. Seu conteúdo é reproduzido a
partir de `src/routes/**` a cada build/dev. Durante a execução da
SCP-012.0.1.2, o plugin removeu automaticamente dez linhas de tipagem
e `declare module` que estavam presentes no blob do baseline `2e038071`:

```text
- import type { getRouter } from './router.tsx'
- import type { startInstance } from './start.ts'
- declare module '@tanstack/react-start' {
-   interface Register {
-     ssr: true
-     router: Awaited<ReturnType<typeof getRouter>>
-     config: Awaited<ReturnType<typeof startInstance.getOptions>>
-   }
- }
```

Composição da remoção:

- dois `import type` (type-only, sem emissão JavaScript);
- um `declare module '@tanstack/react-start'`;
- uma `interface Register`;
- propriedades `ssr`, `router`, `config`.
